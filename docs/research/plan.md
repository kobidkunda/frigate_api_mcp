# Factory Analytics UI Research Plan

## Objectives
- Validate navigation clarity across roles (operator, supervisor, admin)
- Validate review queue efficiency and error recovery
- Validate status comprehension (jobs, cameras, system)
- Validate mobile usability of core flows
- Validate reporting usefulness for supervisors
- Validate configuration safety for admins

## Methods
- Moderated sessions (8–12)
- Unmoderated tasks (15–25)
- Heuristic evaluation (Nielsen, Weinschenk)
- SUS + UMUX-Lite
- Analytics instrumentation review (no PII)

## Success Criteria
- Completion ≥85% across roles; no core task <70%
- SUS ≥75; UMUX-Lite ≥72
- Time and error thresholds met
- No severity-4 a11y/usability issues remain
